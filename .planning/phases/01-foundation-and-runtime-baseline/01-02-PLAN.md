## Trust Boundaries

| Boundary                   | Description                                                                                 |
| -------------------------- | ------------------------------------------------------------------------------------------- |
| operator->runtime commands | Shell scripts invoke API and worker processes with operational privileges.                  |
| worker->queue adapter      | Worker process communicates with queue abstraction that will later execute privileged jobs. |

## STRIDE Threat Register

| Threat ID | Category | Component                         | Disposition | Mitigation Plan                                                                                                                  |
| --------- | -------- | --------------------------------- | ----------- | -------------------------------------------------------------------------------------------------------------------------------- |
| T-01-05   | E        | worker runtime bootstrap          | mitigate    | Keep worker and API startup isolated and explicit, reducing accidental privilege/capability coupling between processes.          |
| T-01-06   | D        | process lifecycle                 | mitigate    | Add deterministic start/stop contract in worker adapter scaffold and test lifecycle transitions.                                 |
| T-01-07   | T        | script execution paths            | mitigate    | Restrict scripts to known entrypoints and avoid hidden side effects (no implicit migration or job execution in startup scripts). |
